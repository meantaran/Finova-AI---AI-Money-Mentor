# 💰 AI Money Mentor (Finova)

AI Money Mentor is a Gen-AI powered personal finance assistant designed to help users manage wealth, optimize taxes, and plan financial independence (FIRE).

---

## 🚀 Features

### 🔐 Authentication

* Secure login UI
* Profile management
* Dark/light mode

### 📊 Dashboard

* Net worth tracking
* Investment insights
* Money Health Score

### 🤖 AI Chat Mentor

* Chat-based financial assistant
* Portfolio suggestions (simulated)

### 📈 FIRE Planner

* Calculates retirement corpus
* Adjustable investment, rate, and time

### 💰 Tax Wizard

* Old vs New regime comparison
* Tax-saving insights

### 👤 Profile Management

* Editable user data
* Avatar generation

---
